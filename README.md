# FİTNESS101 - Fitness Merkezi Web Uygulaması

Modern ve responsive bir fitness merkezi web sitesi. HTML5, CSS3 ve Vanilla JavaScript kullanarak oluşturulmuştur.

## 📋 Proje Özellikleri

### ✨ Anasayfa (Index)
- Hero bölümü
- Özellikler panel sistemi
- Antrenman programları özeti
- İstatistikler
- Call-to-action bölümü

### 📖 Hakkında Sayfası
- Şirket misyonu ve vizyonu
- Neden biz bölümü
- Takım üyelerinin profilleri
- Kuruluş hikayesi

### 💪 Programlar Sayfası
- Tüm antrenman programları
- Kategori filtreleme (Güç, Kardiyovasküler, Esneklik, Kilo Vermek)
- Dinamik program kartları
- Detaylı özellikler listesi

### 💰 Paketler (Pricing) Sayfası
- Üç seviye paket: Başlangıç, Profesyonel, Premium
- Fiyatlandırma bilgileri
- Paket özellikleri
- Sık sorulan sorular (FAQ)

### 📧 İletişim Sayfası
- İletişim formu
- İletişim bilgileri
- Çalışma saatleri
- İletişim kanal bilgileri

### 🛠️ Admin Paneli
- Programlar yönetimi (Ekle/Sil)
- Paketler yönetimi (Ekle/Sil)
- İletişim mesajlarını görüntüleme
- Site ayarları
- Veri yönetimi (Export/Import)
- İstatistikler gösterimi

```
SSFF-02/
├── index.html           # Anasayfa
├── about.html           # Hakkında sayfası
├── programs.html        # Programlar sayfası
├── pricing.html         # Paketler sayfası
├── contact.html         # İletişim sayfası
├── admin.html           # Admin paneli
├── styles.css           # Tüm CSS stilleri
├── script.js            # Ana JavaScript dosyası
```

## 🚀 Başlangıç

1. Klasörü açın ve tüm dosyaları bir dizinde bulundurun
2. `index.html` dosyasını tarayıcıda açın
3. Navigasyon menüsünü kullanarak sayfalar arasında gezinin

## 💻 Teknolojiler

- **HTML5**: Semantal yapı
- **CSS3**: Modern stil ve animasyonlar
- **Vanilla JavaScript**: Dinamik işlevsellik ve DOM manipülasyonu
- **LocalStorage**: Veri saklama
